About Datagram

Datagram was created for the aquisition of the data Bloomberg L. C. has to offer in a convenient, user-friendly way. By allowing the user to snap a shot of a logo or flag Datagram encourages an image to be so much more. An image can now be associated with a large amount of data for the aid of learning and exploring.
